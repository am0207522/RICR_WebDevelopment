CSS is used to style and design to the HTML elements like color, font, background, size etc. When HTML, CSS and JavaScript are connected together:- 1.HTML — creates the structuure of the page (headings, buttons, paragraphs,image,lists) 2.CSS — gives design and style to the page (colors, fonts, layout) 3.JavaScript — makes the page interactive (button click,) Without CSS the page looks very plain and boring. CSS makes the page look good , beautiful and attractive.
